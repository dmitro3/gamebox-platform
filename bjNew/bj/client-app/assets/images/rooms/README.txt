================================================================
房间初始图标池
================================================================

【当前默认图标】
    default-room.png    256x256 · ~150KB · AI 渲染的 3D 金色赌场建筑
                        罗马柱 / 红色绒面拱门 / 红宝石山墙 / 大理石底座

平台给代理开通新房间时，默认就用这张作为初始图标。
代理在后台可以上传自己的图标替换它（替换后的图存在 server 端，
不在这个目录）。

【在代码里怎么引用】
    common.js → DEFAULT_ROOM_ICON: '../../assets/images/rooms/default-room.png'

【想增加更多备选图标】
按主题命名直接放进这个文件夹即可，例如：
    gold-01.png           金色主题
    diamond-01.png        钻石主题
    night-01.png          夜场主题
    crown-01.png          皇冠主题
    ...

【推荐尺寸】
正方形 256×256 像素（PNG，透明背景）
文件大小建议 < 50KB

【放完新图片后怎么用】
告诉我你放了哪些文件，我去改 common.js 里 DEFAULT_ROOMS 的
icon 字段。
