## 统一规格

```
SPRITE SHEET for Chinese classic fruit-machine (玛丽机) CRT TV screen.
Exact equal grid, left→right then top→bottom. Dark/black stage background.
Art direction: designed miniature STAGE SCENE with set dressing, props, lighting cues —
NOT abstract particle rings floating alone. Think game-show reveal / arcade attract mode.
Reserve EMPTY dark area in center (or center+lower) for live HTML overlay (digits / fruit icons / titles).
NO readable text, NO Chinese characters, NO digits, NO score, NO watermark, NO logos.
Premium glossy gold / cyan / red arcade neon, cinematic lighting, refined not childish.
```

## A. idle.png — 4×3=12 · 完整「常驻记分牌 UI」分镜（循环）

整屏就是倍数/总投注记分牌：上 LED 窗叠倍数、下 LED 窗叠总投注；
「倍数」「总投注」牌面画进精灵表。跑马灯循环。禁止 CSS 大段文案布局。

## B. gamble-roll.png — 4×4=16 · 完整「猜大小 UI」分镜（循环）

整屏就是猜大小界面：左「小」牌、右「大」牌、中央金圈空心留给跳动数字。
不是 CSS 写字 + 另加特效底图。

## C. gamble-win.png — 3×4=12 · 猜中揭幕 UI 分镜

幕布/庆祝与小大牌一体设计；中央空心叠最终点数。

## D. gamble-lose.png — 3×4=12 · 未中熄灯 UI 分镜

舞台熄灭合幕一体设计；中央空心叠最终点数。

## E. award-train — 4×4=16 · 循环「开火车」

微型火车头/车厢灯光沿椭圆轨道绕中央展示台飞驰（可不画写实火车，用灯条车厢剪影），
中央空台留给水果图标。像开火车报奖过场。

## F. award-big3 — 4×3=12 · 「大三元」三台点亮

豪华金舞台，左中右三座空置奖台依次点亮聚光，最后三束汇合。
空台留给代码叠水果。

## G. award-small3 — 4×3=12 · 「小三元」三台点亮

同大三元构图，银青冷色调、更精巧的小型奖台。

## H. award-four — 4×3=12 · 「大四喜」四角点亮

四角空置奖台依次点亮再汇到中央空台。翠绿+金。

## I. award-slam — 4×4=16 · 「大满贯」盛大揭幕

全幕拉开、金雨、多层追光、中央巨型空展示台留给分数。

## J. award-luck — 4×3=12 · 「运气」神秘灯台

四叶草/幸运灯笼舞台氛围，中央空台留给 luck 图标。

## K. award-bar — 4×3=12 · 「天门/BAR」撞击揭幕

红金霓虹舞台冲击，中央空位留给 BAR 图标，两侧灯柱点亮。
