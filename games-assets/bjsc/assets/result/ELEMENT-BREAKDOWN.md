# 开奖公布 v9 · 光轨颁奖台（原创）

## 设计理念
不再拼贴 AI 徽章/背景，采用统一的 **玻璃颁奖柱** 视觉：
- 三柱式领奖台（冠军居中加高）
- 每柱：称号 → 官方色球号 → 对应色赛车
- 背景：暗色速度线 + 底部光轨
- 底部：F1 计时条风格数据栏

## 结构
| 区域 | 实现 |
|------|------|
| 头栏 | 品牌名 + 胶囊车号轨 |
| 舞台 | Canvas 原创绘制（无徽章 PNG） |
| 赛车 | AI 母版 HSL 换色 |
| 底栏 | 单行 Ticker 数据 |
