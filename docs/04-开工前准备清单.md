## 三、初始化步骤（M0）
```bash
# 1. 根目录装依赖
pnpm install

# 2. 后端
cd server
cp .env.example .env          # 填 DATABASE_URL / JWT_SECRET / REDIS_URL
pnpm prisma generate
pnpm prisma migrate dev --name init
pnpm prisma:seed              # 平台账户 + 代理等级

# 3. 起后端
pnpm --filter @gamebox/server start:dev

# 4.（迁移阶段）新建前端 Vue 工程
#   pnpm create vite client-app -- --template vue-ts
#   + pinia / vue-router / @capacitor/core @capacitor/cli
#   admin-panel 同理 + element-plus + echarts
```
